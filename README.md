# Lobby Connect

After-hours outsourced front-desk service for hotels — phone routing + tablet video.

## Status

v1 in development. Pilot target: one hotel, end-to-end.

## Stack

Next.js + Vite + Supabase + Twilio + Agora, deployed on Vercel.
